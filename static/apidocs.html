API Documentation

Learn how to use our RESTful API.

GET /api/restaurant

Returns a collection of all the Restaurants.

Example:
GET /api/restaurant

200 OK
[ { restaurant1 }, { restaurant2 }, ..., { restaurantn} ]

POST /api/restaurant

Creates a new Restaurant. Requires authentication.

Example:
POST /api/restaurant
[ {"nome":"Gregor Samsa Bug Food Truck", "email": "gregor@kafkaesque.com", "password":"yourpwgoeshere", "owner": { "name":"Gregor Samsa", "cpf":"11122233344", "birthday":"02-01-1915" } } ]

201 CREATED
[ { "message":"Restaurant Created." } ]

GET /api/restaurant/:key

Returns the restaurant specified by the key parameter.

Example:
GET /api/restaurant/imAgInETHisIsAResTauRanTKEy

200 OK
[ { "key":"imAgInETHisIsAResTauRanTKEy", ..., "name":"Los Pollos Hermanos" } ]

DELETE /api/restaurant/:key

Deletes the restaurant specified by the key parameter. Requires authentication.

Example:
DELETE /api/restaurant/imAgInETHisIsAResTauRanTKEy

200 OK
[ { "message":"Hasta la vista, baby!" } ]

PATCH /api/restaurant/:key

Patches the restaurant specified by the key parameter. Requires authentication. Only replace operations are allowed.

Example:
PATCH /api/restaurant/imAgInETHisIsAResTauRanTKEy
[ {"op":"replace", "path":"/email", "value":"gregor_100sual@hotmail.com"} ]

200 OK
[ { "message":"Yay! Success!" } ]

GET /api/restaurant/:key/menu

Returns a collection of all menu items of the Restaurant specified by the key parameter.

Example:
GET /api/restaurant/imAgInETHisIsAResTauRanTKEy/menu

200 OK
[ { item1 }, { item2 }, ..., {itemn} ]

POST /api/restaurant/:key/menu

Creates a new item in the menu of the Restaurant specified by the key parameter Requires authentication.

Example:
POST /api/restaurant/imAgInETHisIsAResTauRanTKEy/menu
[ {"name":"Vermin au Cordon Bleu", "price":"59.90", "desc":"Splendide!"} ]

201 CREATED
[ { "message":"Yay! Success!" } ]

GET /api/restaurant/:key/menu/:itemkey

Returns the menu item specified by the itemkey parameter from the restaurant specified by the key parameter.

Example:
GET /api/restaurant/imAgInETHisIsAResTauRanTKEy/menu/aKeY

200 OK
[ { "name":"Vermin au Cordon Bleu", "price":"59.90", "description":"Splendide!" } ]

PUT /api/restaurant/:key/menu/:itemkey

Replaces the menu specified by the parameter itemkey in the restaurant specified by the parameter key. Requires authentication.

Example:
PUT /api/restaurant/imAgInETHisIsAResTauRanTKEy/menu/aKeY
[ {"name":"Vermin au Cordon Bleu", "price":"99.90", "desc":"Splendide!"} ]

200 OK
[ { "message":"Yay! Success!" } ]

DELETE /api/restaurant/:key/menu/:itemkey

Deletes the menu item specified by the itemkey parameter from the restaurant specified by the key parameter. Requires authentication.

Example:
DELETE /api/restaurant/imAgInETHisIsAResTauRanTKEy/menu/aKeY

200 OK
[ { "message":"Dead." } ]
